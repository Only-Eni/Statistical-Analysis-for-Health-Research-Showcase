# 📊 Statistical Analysis for Health Research (Confidential Client Projects)

## Overview

This repository highlights my experience providing statistical consulting for health research projects using **IBM SPSS Statistics** and **Microsoft Excel**.

The projects involved end-to-end statistical support, including data preparation, analysis, interpretation, and dissertation-ready reporting.

> **Confidentiality Notice**
>
> To protect client privacy and unpublished academic work, the datasets, SPSS outputs, dissertation chapters, and research findings are intentionally omitted.

## 🔹 Services Delivered

- Data cleaning and validation
- Variable coding and recoding
- Missing data handling
- Composite score computation (Knowledge, Attitude, Practice)
- Descriptive statistics
- Independent Samples *t*-test
- One-Way ANOVA
- Post Hoc Analysis (Tukey / Games-Howell)
- Pearson Correlation
- Regression Analysis (where applicable)
- APA statistical reporting
- Chapter Four preparation
- Statistical interpretation
